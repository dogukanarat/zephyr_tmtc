*# zephyr_tmtc
TMTC API implemented for Zephyr OS

## Overview
This library provides a Telemetry and Telecommand (TMTC) API designed specifically for the Zephyr OS. It facilitates communication between embedded systems ensuring reliable data exchange.

## Features
- Lightweight and efficient implementation
- Communication protocol agnostic
- Easy integration with Zephyr-based projects

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/dogukanarat/zephyr_tmtc.git
   ```
2. Navigate to the project directory:
   ```bash
   cd zephyr_tmtc
   ```
3. Follow the setup instructions in the `prj.conf` file to configure your Zephyr environment.
